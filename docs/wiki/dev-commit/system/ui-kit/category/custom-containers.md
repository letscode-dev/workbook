# Контейнеры: tip, info, warning, danger, details

### Информация
- Custom containers (кастомные контейнеры) — в документации VitePress они описаны именно так: встроенная возможность markdown, а не отдельный Vue-компонент.
В markdown можно использовать блоки `::: тип заголовок` … `:::`.
- https://vitepress.dev/guide/markdown.html#custom-containers

| Тип | Назначение |
|-----|------------|
| `tip` | Подсказки, рекомендации |
| `info` | Информационные заметки |
| `warning` | Предупреждения о возможных проблемах |
| `danger` | Важные/критичные предупреждения |
| `details` | Раскрывающийся блок (клик по заголовку открывает/закрывает контент) |

### Варианты

::: tip Подсказка
Рекомендация или полезный совет.
:::

::: info Информация
Нейтральная заметка или справка.
:::

::: warning Внимание
Предупреждение о возможных проблемах.
:::

::: danger Опасность
Важное предупреждение или критичная информация.
:::

::: details Раскрывающийся блок
Нажмите на заголовок, чтобы развернуть или свернуть содержимое.
:::
