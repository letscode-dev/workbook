# Яндекс Карты и Google Maps

::: details Темы
- Статичные и динамичные карты
- Фиксированный размер
- Размер в процентах
- `scroll=false`
:::

```html:no-line-numbers
<script 
	type="text/javascript" 
	charset="utf-8" 
	async 
	src="
		https://api-maps.yandex.ru/services/constructor/1.0/js/?um=constructor...&amp;
		width=500&amp;
		height=400&amp;
		lang=ru_RU&amp;
		scroll=false
	"
></script>
```
