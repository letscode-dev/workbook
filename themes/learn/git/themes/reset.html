Отмена изменений файлов

  • Варианиы отмены изменений файлов в разных состояниях: Untracked, Modified, Staged и Commited

1. Untracked

git clean -f

2. Modified

git restore index.html  # Только для index.html
git restore .           # Все файлы

3. Staged

git restore --staged .
git reset

4. Commited

git reset HEAD~1
